# Chronoserve

Very simple HTTP file server.

Can be used to:

- Traverse through the directory remotely
- View data
- Test HTML pages

## Usage parameters

```sh
  -folder string
        Where to start the server (default ".")
  -host string
        Hostname to be used to serve the data (default "127.0.0.1")
  -port int
        Port to start listening on (default 8003)
```
